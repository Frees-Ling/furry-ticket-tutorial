# 票务系统教程目录

## 项目概述

从零构建一个完整的票务系统（Ticket Booking System），包含：

- 后端：Python FastAPI + SQLAlchemy 2.0 + MySQL 8.0 + Redis 7.x
- 前端：Vue 3 + Pinia + Vue Router
- 部署：Nginx + Docker + GitHub Actions + ngrok/frp

**适合人群：** Python 基础开发者，想学习全栈开发

**前置知识：** Python 基础语法

## 架构参考

```text
┌─────────┐     ┌──────────┐     ┌───────────┐     ┌─────────────────┐
│ 用户     │ ──→ │ CDN      │ ──→ │ Nginx     │ ──→ │ FastAPI (Uvicorn)│
│ 浏览器   │     │ (静态文件)│     │ 反向代理   │     │ :8000           │
│ (Vue 3) │     └──────────┘     │ SSL 终止   │     └──────┬──────────┘
└─────────┘                      │ 限流       │            │
                                 │ 负载均衡   │     ┌──────┴──────────┐
                                 └───────────┘     │ ARQ Worker      │
                                                   │ (后台任务)      │
                                                   └──────┬──────────┘
                                                          │
                          ┌───────────────────────────────┼───────────────────┐
                          │                               │                   │
                    ┌─────┴─────┐                  ┌──────┴──────┐     ┌────┴────┐
                    │ MySQL 8.0  │                  │ Redis 7.x   │     │ 支付宝  │
                    │ :3306     │                  │ :6379       │     │ API     │
                    │ (持久层)  │                  │ (缓存+库存)  │     │ (支付)  │
                    └───────────┘                  └─────────────┘     └─────────┘
```

## 核心算法

| 算法 | 章节 | 说明 |
| ------ | ------ | ------ |
| Redis Lua 原子库存扣减 | 06 | 防超卖第一层防御，~0.1ms |
| MySQL 乐观锁 | 06 | 防超卖第二层防御，持久化兜底 |
| JWT HMAC-SHA256 | 07 | 令牌签名验证，防篡改 |
| bcrypt 密码哈希 | 07 | 密码存储，GPU 抗性 |
| WebSocket 心跳检测 | 08 | 断线检测，超时关闭 |
| Little 定律 L=λW | 10 | 排队论，系统容量规划 |
| 漏桶限流 | 11 | Nginx 请求限流 |
| RSA2 签名 | 16 | 支付宝支付签名验证 |
| 幂等性检查 | 16 | 支付通知防重放 |
| SMS 验证码 | 37 | 短信验证码发送/校验/60s限制 |
| CAPTCHA 防机器人 | 38 | 一次性令牌+数学题验证 |
| OperationLog 操作审计 | 39 | 操作轨迹记录/分页查询/Tab筛选 |
