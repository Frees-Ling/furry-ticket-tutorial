# Windows 本地服务器部署指南

## 架构概述

```text
┌─────────────────────────────────────────────────┐
│                  Windows Server                  │
│                                                   │
│  ┌──────────┐   ┌──────────┐   ┌──────────────┐  │
│  │ Nginx    │   │ FastAPI  │   │ ARQ Worker   │  │
│  │ (端口80) │──→│ (端口8000)│   │ (后台任务)   │  │
│  └──────────┘   └────┬─────┘   └──────┬───────┘  │
│                      │                 │          │
│  ┌──────────┐        │                 │          │
│  │ Vue3 SPA │        │                 │          │
│  │ (静态文件)│        │                 │          │
│  └──────────┘        ▼                 ▼          │
│              ┌─────────────┐  ┌──────────────┐   │
│              │ MySQL 8.0   │  │ Memurai/Redis│   │
│              │ (端口3306)  │  │ (端口6379)   │   │
│              └─────────────┘  └──────────────┘   │
└─────────────────────────────────────────────────┘
```

**组件说明：**

| 组件 | 角色 | 进程管理方式 |
| ------ | ------ | ------------- |
| Nginx for Windows | 反向代理 + 静态文件服务 | Windows 服务（自动启动） |
| FastAPI (Uvicorn) | Web API 服务 | NSSM 注册为 Windows 服务 |
| ARQ Worker | 后台任务（自动取消、通知） | NSSM 注册为 Windows 服务 |
| MySQL 8.0 | 关系型数据库 | Windows 服务（官方 MSI 安装） |
| Memurai/Redis | 缓存 + 库存 | Windows 服务（Memurai） |
| Vue3 SPA | 前端静态文件 | 由 Nginx 直接 serve |

## 1. 前置条件安装

### 1.1 Python 3.10+

```powershell
# 下载 Python 3.10+ 安装包
# https://www.python.org/downloads/windows/
# 安装时勾选 "Add Python to PATH"

# 验证安装
python --version
pip --version

# 配置国内镜像源（可选）
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```text

### 1.2 MySQL 8.0

```powershell
# 下载 MySQL 8.0 MSI 安装包
# https://dev.mysql.com/downloads/installer/

# 安装选项：
# - Setup Type: Server only
# - 配置端口: 3306
# - 认证方式: Use Strong Password Encryption
# - root 密码: 设置强密码（记录到密码管理器）
# - Windows Service: 勾选 "Start the MySQL Server at System Startup"

# 验证安装
mysql -uroot -p
SHOW VARIABLES LIKE 'version';
```

### 1.3 Memurai（Windows Redis 替代方案）

Memurai 是兼容 Redis 6.x 的 Windows 原生实现：

```powershell
# 下载 Memurai 开发者版（免费）
# https://www.memurai.com/

# MSI 安装后自动注册为 Windows 服务
# 默认端口: 6379

# 验证安装
memurai-cli ping  # 应返回 PONG
memurai-cli info server

# 或者使用 Redis for Windows（由 Microsoft 维护的移植版）
# https://github.com/microsoftarchive/redis/releases
# 下载 Redis-x64-xxx.msi 安装
# 之后注册为服务：redis-server --service-install
```text

### 1.4 NSSM（Non-Sucking Service Manager）

NSSM 将任何可执行文件注册为 Windows 服务，支持自动重启、日志记录：

```powershell
# 下载 NSSM
# https://nssm.cc/download
# 解压到 C:\nssm

# 将 nssm.exe 加入 PATH
[Environment]::SetEnvironmentVariable(
    "Path",
    [Environment]::GetEnvironmentVariable("Path", "User") + ";C:\nssm\win64",
    "User"
)

# 验证
nssm version
```

## 3. 注册 Windows 服务

### 3.1 FastAPI 服务

```powershell
# 安装 FastAPI 为 Windows 服务
nssm install TicketAPI

# NSSM 会打开 GUI 配置界面，填写：
#   Application Path: C:\ticket-booking\venv\Scripts\uvicorn.exe
#   Startup directory: C:\ticket-booking
#   Arguments: app.main:app --host 0.0.0.0 --port 8000 --workers 2 --loop uvloop
#
# 切换到 Details 标签：
#   Display name: Ticket Booking API
#   Startup type: Automatic
#
# 切换到 I/O 标签：
#   Output (stdout): C:\ticket-booking\logs\api_stdout.log
#   Error (stderr): C:\ticket-booking\logs\api_stderr.log
#
# 点击 "Install Service"

# 或使用命令行方式（推荐，可脚本化）：
nssm install TicketAPI `
    "C:\ticket-booking\venv\Scripts\uvicorn.exe" `
    "app.main:app --host 0.0.0.0 --port 8000 --workers 2"

nssm set TicketAPI AppDirectory "C:\ticket-booking"
nssm set TicketAPI DisplayName "Ticket Booking API"
nssm set TicketAPI Start SERVICE_AUTO_START
nssm set TicketAPI AppStdout "C:\ticket-booking\logs\api_stdout.log"
nssm set TicketAPI AppStderr "C:\ticket-booking\logs\api_stderr.log"
nssm set TicketAPI AppRotateFiles 1
nssm set TicketAPI AppRotateOnline 1
nssm set TicketAPI AppRotateBytes 10485760  # 10MB 轮转
```

### 3.2 ARQ Worker 服务

```powershell
# 安装 Worker 为 Windows 服务
nssm install TicketWorker `
    "C:\ticket-booking\venv\Scripts\python.exe" `
    "-m arq app.worker.WorkerSettings"

nssm set TicketWorker AppDirectory "C:\ticket-booking"
nssm set TicketWorker DisplayName "Ticket Booking Worker"
nssm set TicketWorker Start SERVICE_AUTO_START
nssm set TicketWorker AppStdout "C:\ticket-booking\logs\worker_stdout.log"
nssm set TicketWorker AppStderr "C:\ticket-booking\logs\worker_stderr.log"
nssm set TicketWorker AppRotateFiles 1
nssm set TicketWorker AppRotateOnline 1
nssm set TicketWorker AppRotateBytes 10485760
nssm set TicketWorker AppExit Default Exit  # Worker 正常退出后自动重启

# Worker 需要常驻运行，如果进程退出，NSSM 会在 2 秒后自动重启
nssm set TicketWorker AppThrottle 15000  # 15秒内最多重启一次
```text

### 3.3 Nginx

```powershell
# 下载 Nginx for Windows
# https://nginx.org/en/download.html → nginx/Windows-x.y.z.zip
# 解压到 C:\tools\nginx-1.26.0\

# 复制项目配置文件
Copy-Item C:\ticket-booking\nginx\nginx.windows.conf `
    C:\tools\nginx-1.26.0\conf\nginx.conf

# 复制前端构建产物
Copy-Item -Recurse C:\ticket-booking\frontend\dist\* `
    C:\tools\nginx-1.26.0\html\ticket-frontend\

# 注册为 Windows 服务（使用 Windows 自带 SC 命令）
sc create TicketNginx binPath="C:\tools\nginx-1.26.0\nginx.exe -g daemon off;" start=auto displayname="Ticket Booking Nginx"

# 启动 Nginx
Start-Service TicketNginx

# 验证 Nginx
curl http://localhost/health
```

### 3.4 服务管理命令汇总

```powershell
# 启动服务
nssm start TicketAPI
nssm start TicketWorker
Start-Service TicketNginx

# 停止服务
nssm stop TicketAPI
nssm stop TicketWorker
Stop-Service TicketNginx -Force

# 重启服务
nssm restart TicketAPI
nssm restart TicketWorker
Restart-Service TicketNginx

# 查看服务状态
nssm status TicketAPI
Get-Service TicketAPI, TicketWorker, TicketNginx

# 编辑服务参数
nssm edit TicketAPI  # 打开 GUI 编辑
```text

## 4. NSSM 日志管理

### 4.1 日志轮转配置

```powershell
# NSSM 内置日志轮转（已在安装时配置）
# - AppRotateFiles: 启用轮转
# - AppRotateOnline: 运行时轮转（不需要重启服务）
# - AppRotateBytes: 10MB 时轮转

# 日志文件位置：
# C:\ticket-booking\logs\
#   ├── api_stdout.log      # FastAPI 标准输出
#   ├── api_stderr.log      # FastAPI 错误输出
#   ├── worker_stdout.log   # Worker 标准输出
#   └── worker_stderr.log   # Worker 错误输出

# Nginx 日志：
# C:\tools\nginx-1.26.0\logs\
#   ├── access.log
#   └── error.log
```

### 4.2 配置 Windows EventLog 集成

```powershell
# 创建事件源（需要管理员权限）
New-EventLog -LogName Application -Source "TicketBooking"

# 使用 PowerShell 写入事件日志
Write-EventLog -LogName Application -Source "TicketBooking" `
    -EventId 1000 -EntryType Information -Message "服务已启动"

# Python 应用中写入 Windows EventLog（在 app/logging_config.py 中配置）
# 安装 pip install pywin32
# 使用 win32evtlog 模块写入
```text

### 4.3 使用 Windows 任务计划程序自动清理旧日志

```powershell
# 创建日志清理任务
$action = New-ScheduledTaskAction -Execute "powershell.exe" `
    -Argument "-Command Remove-Item 'C:\ticket-booking\logs\*.log.*' -Force"

$trigger = New-ScheduledTaskTrigger -Daily -At 02:00

Register-ScheduledTask -TaskName "CleanTicketLogs" `
    -Action $action -Trigger $trigger `
    -RunLevel Highest -User "SYSTEM"
```

## 5. 备份策略

### 5.1 MySQL 数据库备份

```powershell
# 创建备份目录
$backupDir = "C:\backups\ticket-booking"
New-Item -ItemType Directory -Path $backupDir -Force | Out-Null

# 执行备份
$timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
$backupFile = "$backupDir\ticket_dev_$timestamp.sql"

mysqldump -uroot -p你的密码 `
    --databases ticket_dev `
    --add-drop-database `
    --routines `
    --triggers `
    --single-transaction `
    --result-file=$backupFile

# 压缩备份
Compress-Archive -Path $backupFile -DestinationPath "$backupFile.zip"
Remove-Item $backupFile

# 保留最近 30 天的备份，删除更早的
Get-ChildItem $backupDir -Filter "*.zip" |
Where-Object { $_.LastWriteTime -lt (Get-Date).AddDays(-30) } |
Remove-Item -Force
```text

### 5.2 设置定时备份任务

```powershell
# 创建备份脚本
$psScript = @"
`$backupDir = "C:\backups\ticket-booking"
New-Item -ItemType Directory -Path `$backupDir -Force | Out-Null
`$timestamp = Get-Date -Format "yyyyMMdd_HHmmss"
`$backupFile = "`$backupDir\ticket_dev_`$timestamp.sql"
mysqldump -uroot -p你的密码 --databases ticket_dev --single-transaction --result-file=`$backupFile
Compress-Archive -Path `$backupFile -DestinationPath "`$backupFile.zip" -Force
Remove-Item `$backupFile
# 清理 30 天前的旧备份
Get-ChildItem `$backupDir -Filter "*.zip" | Where-Object { `$_.LastWriteTime -lt (Get-Date).AddDays(-30) } | Remove-Item -Force
"@

Set-Content -Path "C:\ticket-booking\scripts\backup.ps1" -Value $psScript

# 创建定时任务（每天凌晨 3 点执行）
$action = New-ScheduledTaskAction -Execute "powershell.exe" `
    -Argument "-NoProfile -File C:\ticket-booking\scripts\backup.ps1"

$trigger = New-ScheduledTaskTrigger -Daily -At 03:00

Register-ScheduledTask -TaskName "TicketBookingBackup" `
    -Action $action -Trigger $trigger `
    -RunLevel Highest -User "SYSTEM" `
    -Description "每日备份票务系统数据库"
```

## 8. 安全加固

### 8.1 端口安全

```powershell
# Windows 防火墙：只开放 80 端口
netsh advfirewall firewall add rule name="Allow HTTP" dir=in action=allow protocol=TCP localport=80

# 禁止外部访问 8000（只允许本机 + Nginx）
netsh advfirewall firewall add rule name="Deny Direct API" dir=in action=block protocol=TCP localport=8000 remoteip="any"
netsh advfirewall firewall add rule name="Allow Local API" dir=in action=allow protocol=TCP localport=8000 remoteip="127.0.0.1"
```

### 8.2 文件权限

```powershell
# 限制 .env 文件的访问权限
icacls .env /inheritance:r /grant "SYSTEM:F" /grant "BUILTIN\Administrators:F"
```text
