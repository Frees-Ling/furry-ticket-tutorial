# 🐾 兽聚票务系统

  一站式兽聚活动票务解决方案 —— 从购票到入场，让每场兽聚都井然有序。

  ---

  ## 关于本系统

  **兽聚票务** 是一套专为兽人爱好者聚会/展会活动设计的完整票务管理系统，涵盖活动发布、在线购票、支付对接、验票入场、后台管理全流程。

  系统已在多个兽聚活动中实际部署运营，经历过高并发抢票、防超卖、支付宝支付等真实场景考验。

  ---

  ## 核心功能

  ### 🎫 活动管理
  - 活动创建、发布、排期管理
  - 多票种设置（普通票/VIP票/套票）
  - 活动场次、场馆管理
  - 购票限购策略（每人限购张数、身份校验）

  ### 💳 在线购票
  - 实时选座 + 防超卖机制（Redis + MySQL 双层保障）
  - 支付宝支付集成（沙箱/正式环境）
  - 订单自动取消 + 座位释放（倒计时未支付）
  - 国际支付支持（多币种扩展）

  ### ✅ 验票入场
  - 二维码验证核销
  - 身份信息绑定、实名制购票
  - 防伪码 + 核销守卫

  ### 📊 后台管理
  - 销售仪表盘（实时数据 + 趋势图）
  - 订单管理、退款处理
  - 优惠券/折扣规则系统
  - 会员体系 + 积分系统

  ### 🔐 安全保障
  - JWT 认证 + RBAC 权限控制
  - 全链路风控（设备指纹、IP限流、行为分析）
  - 敏感数据字段级加密
  - 安全审计日志

  ## 技术栈

  | 层级 | 技术 |
  |------|------|
  | 后端框架 | FastAPI + SQLAlchemy 2.0 + Pydantic v2 |
  | 数据库 | MySQL 8.0 + Redis 7.x |
  | 消息/任务 | WebSocket + SSE + ARQ 任务队列 |
  | 支付 | 支付宝（沙箱/生产环境） |
  | 前端 | Vue 3 + Pinia + Vue Router |
  | 部署 | Docker + Nginx + GitHub Actions CI/CD |
  | 穿透 | ngrok / frp |
  | 测试 | pytest（1300+ 单元测试，覆盖率 >80%） |
